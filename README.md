# École

École — прототип единого рабочего пространства для частных преподавателей и небольших образовательных команд.

Платформа помогает вести учеников и группы, планировать занятия, хранить учебные материалы, проверять домашние задания и общаться с учениками в одном интерфейсе. Проект рассчитан на преподавателей разных направлений: языков, программирования, дизайна, 3D и других дисциплин.

## Возможности

- обзор расписания и задач на сегодня;
- недельный календарь занятий;
- карточки учеников и учебных групп;
- индивидуальные программы обучения;
- подготовка уроков и прикрепление материалов;
- очередь домашних заданий и обратная связь;
- сообщения ученикам и группам;
- редактирование профиля преподавателя;
- адаптивный интерфейс для разных размеров экрана.

## Технологии

- HTML5;
- CSS3;
- JavaScript;
- GSAP для интерфейсных анимаций.

Проект не требует сборщика и может запускаться как обычный статический сайт.

## Текущий статус

Сейчас École является интерактивным прототипом с подключённой регистрацией и авторизацией Supabase. Сессия пользователя и данные профиля преподавателя сохраняются в облаке. Ученики, уроки, задания и сообщения пока используются как демонстрационные данные.

Для полноценной работы с несколькими пользователями планируется дополнительно подключить:

- облачную базу данных;
- раздельное хранение данных каждого преподавателя;
- файловое хранилище для аватаров и учебных материалов;
- синхронизацию данных между устройствами.

В качестве серверной платформы может использоваться Supabase: Auth для регистрации и входа, PostgreSQL для данных и Storage для файлов.

## Структура проекта

```text
index.html          Основная разметка приложения
styles.css          Стили и адаптивное оформление
app.js              Интерактивность и логика интерфейса
vendor/gsap.min.js  Локальная библиотека анимаций
```

## Запуск

Откройте `index.html` в современном браузере.

## Примечание

В репозитории не должны храниться пароли, секретные ключи или персональные данные учеников.
